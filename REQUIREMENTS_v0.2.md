# BPRecorder v0.2 需求文档

## v0.2 目标

模块化改造 + 多用户预留框架

---

## 一、模块化

### 1.1 识别模块

#### 布局调整
- 右上角显示当前识别方式（如"前端听写"）
- "听写"按钮改为"识别"按钮
- 描述改为："支持直接输入和当前选择方式"
- 当前方式显示通过函数调用关联到识别按钮和配置区的选择项

#### 识别按钮行为
- 语音识别：显示"识别"+"停止"两个状态
- 蓝牙/拍照：只有"识别"（一次性动作）
- 点击"识别" → 执行当前选中的识别方式

#### 配置方式
- **长按**识别区标题 → 进入识别配置区
- 可选识别方式：
  - 前端听写（当前实现，Apple/微软自动切换）
  - 云端听写（预留）
  - 蓝牙识别（预留）
  - 拍照识别（预留）

#### 配置区内容
- 当前方式选择
- 云端听写：API类型(飞书/通义/Whisper) + URL + Token
- 蓝牙：系统配对设备列表 + 选择
- 拍照：暂空

#### 模块化目标
- 新增识别方式 → 只需增加新模块代码 + 修改配置区
- 不改动主逻辑

### 1.3 存储模块（记录区）

#### 当前硬编码位置
- 文件：index.html
- 代码：line 715 `const text = \`${item.time}---${item.high}/${item.low}(${item.diff})-${item.pulse}🩺\`;`
- 需改造为可配置的格式化字符串

#### 记录区格式
- 可自定义格式化字符串
- 示例：
  - `HH:MM---高压/低压(压差)-心率🩺` → `23:45---120/80(40)-60🩺`
  - `HH:MM---高压/低压-压差-心率🩺` → `23:45---120/80-40-60🩺`
  - `HH:MM---高压/低压心率🩺` → `23:45---120/8060🩺`
- 压差显示/隐藏、压差符号 `()` 或 `/` 都在格式字符串中体现

#### 备注处理
- 备注显示在记录区最下方一行
- 复制按钮只复制正式内容，不含备注

#### 配置方式
- **长按**记录区标题 → 进入记录配置区

#### 当前状态
- 刷新后记录消失（纯内存）
- 预留云端存储选项

#### 模块化目标
- 新增存储方式 → 只需增加新模块代码 + 修改配置区
- 不改动主逻辑

## 二、多用户系统

### 2.1 用户状态

#### 未注册/未登录

- 无云端存储

- 预留框架

## 三、模块化架构设计

### 3.1 模块接口

```javascript
// 识别模块接口
class RecognitionModule {
    name           // 模块名称
    start()       // 开始识别
    stop()        // 停止识别
    onresult      // 识别结果回调
}

// 提取模块接口  
class ExtractionModule {
    name           // 模块名称
    extract(text) // 从文本提取数据
}

// 存储模块接口
class StorageModule {
    name           // 模块名称
    save(record)  // 保存记录
    load(date)    // 加载记录
    list()        // 列出记录
}

// 认证模块接口
class AuthModule {
    login(user, pass)      // 登录
    register(user, pass, email)  // 注册
    logout()                // 登出
    resetPassword(email)    // 重置密码（预留）
    isLoggedIn()            // 是否已登录
    getCurrentUser()        // 获取当前用户
}
```

### 3.2 模块注册

```javascript
// 模块注册表
const modules = {
    recognition: {
        current: 'browser',  // 当前使用的识别模块
        browser: new BrowserRecognitionModule(),
        cloud: new CloudRecognitionModule(),    // 预留
        bluetooth: new BluetoothModule(),       // 预留
        camera: new CameraModule()              // 预留
    },
    extraction: {
        current: 'regex',
        regex: new RegexExtractionModule(),
        model: new ModelExtractionModule()     // 预留
    },
    storage: {
        current: 'none',
        none: new NoneStorageModule(),          // 当前无存储
        cloud: new CloudStorageModule()         // 预留
    },
    auth: new AuthModule()
}
```
